Thay đổi v2
→Nav tách 2 nút. Nút 1: con trỏ + kéo giấy (pointer). Nút 2: vùng chữ nhật + lasso (selection). Mỗi nút ▸ flyout riêng.
→Loại bút cả 2 mode. Gõ: ngòi + nhớ (bỏ tẩy). Bút: ngòi + nhớ + tẩy. Luôn hiện, cùng animation transition.
→Context gọn. Chỉ giữ Convert (⇄). Bỏ Copy · Nhân đôi · Xoá — chuyển sang menu ngữ cảnh long-press hoặc chuột phải.
→Thu gọn mới. Nút ═ ở đáy. Trạng thái gọn: hình vuông 56×56px bo 16px, hiện icon bút active. Bấm vào bung lại.
